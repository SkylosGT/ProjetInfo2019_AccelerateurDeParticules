ACCELERATEUR DE PARTICULES :

Informations essentielles

1. Syntax Guidlines

"Sauf informations contraires dans les exercices, 
les règles de syntaxe suivantes seront observées"

Variables/Attributs :
		
	I. Variables/Attributs scalaires <int, double, long, ...> : "scal_nom" (le nommage par une lettre est autorisé si il est suivi 
	   par un commentaire d'indication, ex : scal_x //postion sur l'axe x)

	II. Variables/Attributs vectorielles<Vecteur3D> : "vec_nom" (le nommage par une lettre est autorisé si il est suivi 
	   par un commentaire d'indication, ex : vec_r //vecteur postion)

!!Provisoire!!	//III. Variables/Attributs charactère: "texte_nomPrenom" (le nom se doit d'être explicite, ex : texte_retourEcran)

Passage/Référence :
	I. Variables/Attributs : "_nom" (le nommage par une lettre est autorisé si il reprend 
	   une variable/attribut déjà nommée sinon le nom se doit d'être explicite , ex : _x)
